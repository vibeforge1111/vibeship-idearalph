# /hackathon Page Design Spec

## Target Audience
Vibe coders, indie hackers, and Seedify community members who want to build startups fast using AI tools. They value speed, fun, and real rewards.

## Design System (Matching Existing Site)

### Colors (from existing site)
```
ralph-yellow: #FFD93D      // Primary accent, CTAs
playground-green: #6BCB77  // Success, positive actions
sky-blue: #4D96FF          // Info, secondary accent
playground-sunset: #FF6B6B // Background gradients
playground-orange: #FFA94D // Warning, highlights
chalkboard: #2D3436        // Text, borders
paper: #FFF9E6             // Card backgrounds
white: #FFFFFF             // Clean backgrounds
```

### Typography (existing)
```
font-chalk: "Fredoka One", cursive  // Headings, emphasis
font-body: System UI                 // Body text
```

### Components (existing patterns)
```
Borders: border-3 or border-4 border-chalkboard
Shadows: shadow-crayon, shadow-crayon-lg
Buttons: btn-crayon class (yellow bg, black border, shadow, hover transforms)
Cards: rounded-xl or rounded-2xl, border-2 border-chalkboard/20
```
